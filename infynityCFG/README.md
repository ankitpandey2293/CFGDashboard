# Infynity Material Dashboard

## Getting started


Install dependencies:

    $ cd infynityCFG
    $ npm install

Install gem 'sass'

    $ gem install sass
    
Run development web-server:

    $ gulp serve

## Features

* AngularJS
* Angular UI Router
* Angular Material
* Sass styles
* Gulp build
* Stylish, clean, responsive layout with original design
* BrowserSync for the ease of development
